# inbox/

**1日1ファイル、YYYY-MM-DD.md**。スマホから即書ける入口。

## 書き方

- 気持ち/違和感/観察/アイデアを、フォーマットを気にせず1行ずつ放り込む
- 分類は Claude が翌朝に `daily/` へ整理する
- タグは自然に書けば拾われる（例: `#違和感` `#work` `#idea`）

## 例

```
- 09:12 シフト管理、田口さんが結局スプシに戻ってる。UIじゃなくて通知が課題？ #違和感 #work
- 12:40 家計ビューア、月末しか見てない。日次で開かせる仕掛けが無い #kakei
- 21:15 「AIに任せる/任せない」の線引きを、他のメンバーに言語化して伝えたい #idea
```
